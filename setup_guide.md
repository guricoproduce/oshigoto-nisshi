# LINEミニアプリ お仕事報告システム セットアップガイド

このフォルダには、こども万博向けの「お仕事体験報告＆デジタル認定証発行システム」のMVPコードが格納されています。

## フォルダ構成
- `index.html`: フロントエンド（LIFF）
- `backend.gs`: バックエンド（Google Apps Script）
- `setup_guide.md`: このマニュアル

## 1. LINE Developers の設定
1. [LINE Developers](https://developers.line.biz/) でプロバイダーを作成。
2. **Messaging API** チャネルを作成。
   - 「チャネルアクセストークン（長期）」を発行し、メモ。
3. **LINE Front-end Framework (LIFF)** チャネルを同じプロバイダー内に作成。
   - 「LIFF ID」を発行し、メモ。

## 2. Google スプレッドシートの準備
1. 新規スプレッドシートを作成。
2. ブラウザのURLから「スプレッドシートID」をメモ。
3. 1行目に見出し（タイムスタンプ, UserId, なまえ, お仕事, 感想）を入力。

## 3. Google Apps Script (GAS) のデプロイ
1. スプレッドシートのメニューから「拡張機能」>「Apps Script」を選択。
2. `backend.gs` の内容を貼り付け。
3. 各種ID（Token, SheetID等）を書き換え。
4. 「デプロイ」>「新しいデプロイ」> 種類：ウェブアプリ > 全員に公開。
5. 発行された「ウェブアプリURL」をメモ。

## 4. 認定証テンプレートの作成
1. **Google Slidesを作成**: 横向き（A4サイズなど）で表彰状のデザインを作成。
2. **プレースホルダーを配置**:
   - 名前を入れたい場所に `{{name}}`
   - お仕事名を入れたい場所に `{{job}}`
   と入力（フォントや色、配置は自由）。
3. **IDをメモ**: スライドのURLから「スライドID」をメモ。
4. **保存用フォルダ**: 画像化したファイルを一時保存するDriveフォルダを作成し、その「フォルダID」をメモ。

## 5. Google Drive API の有効化（重要）
1. GASエディタの左メニュー「サービス」の横の `+` をクリック。
2. **Drive API** を選択して「追加」ボタンを押す。
   - これを行わないと、画像のサムネイルリンク取得でエラーになります。

## 6. フロントエンド（LIFF）の公開
1. `index.html` を GitHub Pages や Vercel 等に公開。
   - `LIFF_ID` と `GAS_URL`（ウェブアプリURL）を書き換え。
2. LINE Developers の LIFF設定で「Endpoint URL」に登録。

## 7. 動作確認
1. LIFFのURLをLINEで開き、送信。
2. スプレッドシートに記録され、数秒後に「名前入りの認定証」が画像で届けば成功！
